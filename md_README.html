Jogo

O jogo consiste de um macrotabuleiro de jogo da velha formado por microtabuleiros:

    SUPER JOGO DA VELHA

   1 2 3 | 4 5 6 | 7 8 9
         |       |    
1  · · · |       | · X X  1
2  · · O |   X   | X · ·  2
3  X O · |       | · · O  3
---------+-------+---------
4  · · · |       |        4
5  · O · |   O   |   O    5
6  · O · |       |        6
---------+-------+---------
7  X · · | O X · | · · X  7
8  · · · | · · · | O · ·  8
9  O X · | · · X | X · ·  9
         |       |    
   1 2 3 | 4 5 6 | 7 8 9

Ao vencer um microtabuleiro, o jogador ganha um ponto no macrotabuleiro, como visto acima em 3 deles. Para vencer a partida, basta vencer o jogo da velha no macrotabuleiro. Um empate no macrotabuleiro é possível, uma vez que um empate no microtabuleiro não altera o jogo. Existem 3 tipos de jogadores distintos:

  • Humano: as jogadas são digitadas no teclado do computador, direto no terminal
  • Estabanado: ele sempre joga numa posição aleatória do macrotabuleiro
  • Come-crú: ele sempre escolhe a primeira posição livre do macrotabuleiro

Além disso, há dois modos de jogo:

  • Convencional: Um jogador joga após o outro, na ordem convencional.
  • Aleatório: A cada rodada, um jogador aleatório é sorteado para jogar.

Testes

Ao rodar o programa, você verá a opção de rodar testes automatizados. Os testes incluem:

  • Testes unitários, realizados com Pytest
  • Testes do jogo completo, simulando partidas inteiras
  • Porcentagem de cobertura do código (realizado em conjunto com os anteriores)

Dependências

  • Python 3
  • Pytest
  • Coverage

Você só precisa instalar as duas últimas caso queira rodar os testes automatizados.

As dependências podem ser instaladas no Ubuntu com:

$ sudo apt install python3
$ sudo pip3 install pytest coverage

Como utilizar

Primeiro, clone esse repositório onde desejar com o comando

$ git clone https://github.com/Guilhermev25/MAC0216-EP4

Em seguida, basta rodar o arquivo principal com os comandos

$ sudo chmod a+x RunGame.sh
$ ./RunGame.sh

Uma interface será aberta no terminal, onde você poderá escolher quais ações o programa executará.
